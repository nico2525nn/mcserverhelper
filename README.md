# McServerHelper

Minecraftサーバーの管理を簡単にするためのWeb UIツールです。
サーバーの起動・停止、コマンド実行、バックアップ、Mod/Pluginの管理などをブラウザから直感的に行えます。
また、[ownserver](https://ownserver.kumassy.com/)との連携により、ポート開放なしでサーバーを外部に公開することも可能です。

![screenshot](Screenshot.png)

## 主な機能

- **サーバー管理**: Web UIからMinecraftサーバーの起動・停止、メモリ割り当ての設定が可能です。
- **Webコンソール**: サーバーのコンソールがリアルタイムに表示され、コマンドを直接送信できます。
- **サーバープロパティ編集**: `server.properties`の内容をWeb UI上から分かりやすい説明付きで編集できます。
- **Modrinth連携**:
    - **Mod/Plugin検索**: ModrinthデータベースからModやPluginを名前、Minecraftバージョン、Modローダーで検索できます。
    - **ワンクリックインストール**: 検索結果からバージョンを選択し、ボタン一つでサーバーにインストールできます。
    - **更新チェック**: インストール済みのMod/Pluginに新しいバージョンがないかを確認し、簡単に更新できます。
- **サーバーファイルダウンロード**: Vanilla,Forge,NeoForge,Fabric,Paper,Purpur,Mohistのサーバーファイルを簡単にダウンロードできます
- **ファイル管理**: ローカルからのMod/Pluginのアップロードや、インストール済みファイルの削除が可能です。
- **バックアップ**: ワールドデータのバックアップを簡単に作成・復元できます。
- **ポート開放不要**: ownserver連携機能により、面倒なポート開放作業なしでサーバーやWeb UIを外部に公開できます。

# 使い方

`MCServerHelper.exe`（または`mcserverhelper.py`）を、サーバーファイル（`server.jar`など）と同じフォルダに置いて実行します。

自動的にブラウザで管理画面が開きます。あとはWeb UIの指示に従ってサーバーを管理してください。
ModやPluginの管理も、すべてWeb UI上から行えます。
